# rv32

The `rv32` profile models 32-bit RISC-V with the same generic kernel module
contracts as `rv64`.

Implementation: `src/arch/rv32/ops.cpp`.

The profile provides `stvec`/PLIC/SBI metadata, `ecall` syscall metadata, `sret`
transition metadata, and guarded RV32 inline assembly for stable 64-bit cycle
counter reads, fences, SIE control, and wait/yield behavior.

## Bring-up Plan

- Interrupts: trap vector plus PLIC/CLINT or SBI abstraction.
- Memory: Sv32 page tables.
- User mode: `sret`.
- Syscalls: `ecall`.
- QEMU: `qemu-riscv32` for user-mode debug tests.
