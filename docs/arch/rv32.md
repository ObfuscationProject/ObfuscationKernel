# rv32

The `rv32` profile models 32-bit RISC-V with the same generic kernel module
contracts as `rv64`.

## Bring-up Plan

- Interrupts: trap vector plus PLIC/CLINT or SBI abstraction.
- Memory: Sv32 page tables.
- User mode: `sret`.
- Syscalls: `ecall`.
- QEMU: `qemu-riscv32` for user-mode smoke.
