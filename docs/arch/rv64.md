# rv64

The `rv64` profile models 64-bit RISC-V supervisor mode with 4 KiB pages.

## Bring-up Plan

- Interrupts: trap vector, PLIC for external interrupts, CLINT/SBI timer path.
- Memory: Sv39 first, optional Sv48 later.
- User mode: `sret` from supervisor mode to user mode.
- Syscalls: `ecall` from user mode.
- QEMU: `qemu-riscv64` for user-mode smoke, `qemu-system-riscv64` for trap and
  page-table tests.
