# x86_64

The `x86_64` profile models long mode with 4 KiB base pages, little-endian
memory, and ring 3 user mode.

## Bring-up Plan

- Interrupts: IDT, IST stacks for fault isolation, x2APIC-capable controller.
- Memory: four-level paging first, optional LA57 later.
- User mode: `sysret`/`iretq` transition with canonical-address validation.
- Syscalls: `syscall` fast path with MSR setup and `sysret` return.
- QEMU: `qemu-x86_64` for user-mode smoke, `qemu-system-x86_64` for hardware
  entry tests.
