# x86_64

The `x86_64` profile models long mode with 4 KiB base pages, little-endian
memory, and ring 3 user mode.

Implementation:

- `src/arch/x86_64/ops.cpp`: architecture operations.
- `src/arch/x86_64/boot16.S`: BIOS-loaded first-stage kernel boot sector.
- `src/arch/x86_64/boot.S`: protected-mode to long-mode entry and bootstrap
  page tables.
- `src/arch/x86_64/linker.ld`: freestanding kernel image layout.

The profile provides IDT/IST/x2APIC metadata, `syscall`/`sysret` metadata,
`iretq`/`sysret` transition metadata, and guarded x86-64 inline assembly for
`rdtsc`, `mfence`, interrupt control, and halt/yield behavior.
